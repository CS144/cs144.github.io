const uint64_t  index,

Receives a substring and writes any newly contiguous bytes into the stream.

If accepting all the data would overflow the capacity of this StreamReassembler, then only the part of the data that fits will be accepted. If the substring is only partially accepted, then the eof will be disregarded.

Parameters
datathe string being added
indexthe index of the first byte in data
eofwhether or not this segment ends with the end of the stream

This function accepts a substring (aka a segment) of bytes, possibly out-of-order, from the logical stream, and assembles any newly contiguous substrings and writes them into the output stream in order.
